# map-data-validation-urdu-mumbai# Map Data Verification Project (Mumbai)

## About

I created this project to understand how business data on maps can be checked and verified. I worked with restaurant listings from different areas of Mumbai and tried to check how accurate the information is across sources.

## What I did

I collected 30+ restaurant listings from areas like Mohammad Ali Road, Bhendi Bazaar, Kurla, and Bandra. For each business, I checked:

* Address
* Opening hours
* Business name

I compared Google Maps data with other sources like search results and listings to verify accuracy.

## Urdu data handling

Since many businesses in areas like Mohammad Ali Road and Bhendi Bazaar use Urdu, I also checked how names are written in Urdu and how they appear in Roman English.

I noticed small but important differences like:

* "Bazar" vs "Bazaar"
* "Chowk" vs "Chawk"

This helped me understand how language differences can affect data quality.

## What I found

Not all data is accurate.

Some places had different timings across platforms. In local areas, I also found inconsistencies in naming, especially when Urdu names were converted into English.

## Tools used

* Google Maps
* Google Search
* Excel

## Why I made this

I wanted to practice real-world data verification, especially for roles where both data accuracy and language understanding (like Urdu) are important.
